In some cases, you want a function to modify the actual value of the variable, not just a copy of it. This is done by passing a reference to the variable using pointers. When you pass a variable by reference, you are passing the memory address of the variable instead of its value.

File 4: fac.c - Recursion in C

Recursion Basics

Recursion is a powerful programming technique where a function calls itself to solve a problem. Every recursive function has two key parts:

  1. Base Case: The condition that stops the recursion and prevents infinite loops.
  2. Recursive Case: The function calls itself with modified arguments, gradually reducing the problem size.

Let’s take a look at an example where we calculate the factorial of a number using recursion.

What is Factorial?

The factorial of a number n is the product of all positive integers less than or equal to n. For example:

  • 5! = 5 × 4 × 3 × 2 × 1 = 120
  • 0! = 1 (by definition)

Recursive Function for Factorial

c
int factorial(int n) {
    if (n == 0) {  // Base case
        return 1;
    } else {
        return n * factorial(n - 1);  // Recursive call
    }
}

int main() {
    int result = factorial(5);
    printf("Factorial of 5 is: %d\n", result);  // Output: 120
    return 0;
}
  • Base case: if (n == 0) stops the recursion and returns 1.
  • Recursive case: n * factorial(n - 1) reduces the problem size by 1 and calls the function again.

The function keeps calling itself until n becomes 0, at which point it stops and returns the result back through the chain of calls.

File 6: multpar.c - Matrix Multiplication

Matrix multiplication is more complex than addition. To multiply two matrices, you take the dot product of rows from the first matrix and columns from the second matrix.

Matrix Multiplication Basics

To multiply two matrices, the number of columns in the first matrix must equal the number of rows in the second matrix. The result is a new matrix where each element is the sum of products of corresponding elements.

Example of Matrix Multiplication

Let’s multiply two 2x2 matrices step by step:

c
int main() {
    int matrix1[2][2] = { {1, 2}, {3, 4} };
    int matrix2[2][2] = { {5, 6}, {7, 8} };
    int result[2][2];  // Result matrix

    // Matrix multiplication
    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 2; j++) {
            result[i][j] = 0;  // Initialize result matrix element to 0
            for (int k = 0; k < 2; k++) {
                result[i][j] += matrix1[i][k] * matrix2[k][j];  // Dot product
            }
        }
    }

    // Print the result matrix
    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 2; j++) {
            printf("%d ", result[i][j]);
        }
        printf("\n");
    }

    return 0;
}

The key here is the third loop (over k), which calculates the dot product of rows from matrix1 and columns from matrix2. This is done for each element of the resulting matrix.

Linked Lists in C

A linked list is a dynamic data structure consisting of nodes. Each node contains two parts:

  1. Data: The actual value the node holds.
  2. Next pointer: A pointer that points to the next node in the list.

Linked lists are useful for scenarios where you need dynamic memory allocation, efficient insertion/deletion, or where array sizes are not known beforehand.

1. Node Structure

In C, a node is typically defined as a structure. Each node contains data and a pointer to the next node.

Node Definition:

c
struct Node {
    int data;               // Data field
    struct Node *next;      // Pointer to the next node
};

Here:

  • data: The value the node holds.
  • next: A pointer to the next node in the list (or NULL if it’s the last node).

2. Creating a Linked List

To create a linked list, you need to:

  1. Dynamically allocate memory for a new node.
  2. Initialize the data and next pointer.
  3. Link nodes together by setting the next pointer of one node to the address of the next.

Example:

c
#include <stdio.h>
#include <stdlib.h>

struct Node {
    int data;
    struct Node *next;
};

int main() {
    // Creating the head node
    struct Node *head = (struct Node *)malloc(sizeof(struct Node));
    head->data = 10;
    head->next = NULL;

    // Creating a second node
    struct Node *second = (struct Node *)malloc(sizeof(struct Node));
    second->data = 20;
    second->next = NULL;

    // Linking the first node to the second
    head->next = second;

    printf("First Node Data: %d\n", head->data);
    printf("Second Node Data: %d\n", second->data);

    return 0;
}

In this example:

  • We create two nodes (head and second).
  • head->next = second; links the first node to the second node.
  • Each node is dynamically allocated using malloc.

3. Traversing a Linked List

To traverse a linked list, you start from the head node and move through the list by following the next pointers until you reach the end (where next is NULL).

Traversing Code:

c
struct Node *temp = head;  // Start from the head
while (temp != NULL) {
    printf("%d ", temp->data);  // Print the data
    temp = temp->next;          // Move to the next node
}

In this loop:

  • temp starts at the head of the list.
  • The loop continues until temp is NULL, which indicates the end of the list.

4. Inserting a Node

There are several ways to insert a node into a linked list:

  1. At the beginning of the list.
  2. At the end of the list.
  3. In the middle of the list (after a specific node).

Inserting at the Beginning:

To insert a node at the beginning, you:

  1. Create a new node.

5. Deleting a Node

To delete a node, you:

  1. Find the node to delete.
  2. Update the previous node’s next pointer to skip the deleted node.
  3. Free the memory of the deleted node.

Deleting a Node by Value:

c
void deleteNode(struct Node **head_ref, int key) {
    struct Node *temp = *head_ref, *prev;

    // If the node to be deleted is the head node
    if (temp != NULL && temp->data == key) {
        *head_ref = temp->next;  // Change head
        free(temp);              // Free old head
        return;
    }

    // Find the node to be deleted
    while (temp != NULL && temp->data != key) {
        prev = temp;
        temp = temp->next;
    }

    // If the key was not present in the list
    if (temp == NULL) return;

    // Unlink the node from the linked list
    prev->next = temp->next;
    free(temp);  // Free memory
}
